Set Up Environment

（我本机是Windows10，最后选择使用 WSL + vscode 进行开发）

简要记录下搭建环境的过程：

  1. 安装 WSL Install WSL | Microsoft Learn （默认在系统盘，建议转移到非系统盘）
  2. 在 WSL 中安装并配置 git，下载 clang12
  3. 在本机中下载安装 vscode
  4. 在 WSL 的 /home/ 目录下按照bustub readme中的clone步骤 完成项目clone
  5. 在 WSL 项目里用 code . 命令打开项目
  6. 在 vscode中安装remote-ssh、cmake、cmake tools、clangd、c/c++等插件，详见大佬博客

Testing
